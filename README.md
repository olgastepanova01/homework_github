# Проект "Виджет банковских операций"

## Описание:

Проект "Виджет" - это веб-приложение на Python для визуализации банковских операций клиента.

## Установка:

1. Клонируйте репозиторий посредством команды:
```
git clone  
```

2. Установите зависимости посредством команды: 
```
pip install -r requirements.txt
```

3. Создайте базу данных и выполните миграции посредством команды:
```
python manage.py migrate
```

4. Запустите локальный сервер посредством команды:
```
python manage.py runserver
```

## Использование:

1. Перейдите на страницу Вашего приложения,запустите приложение.
2. Приложение работает успешно, если:
- функция принимает список словарей и опционально значение для ключа 
state (по умолчанию 'EXECUTED') и возвращает новый список словарей, содержащий только те, у которых ключ соответствует указанному значению;
- функция принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание) и возвращает новый список, отсортированный по дате.
