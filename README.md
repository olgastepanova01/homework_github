# Проект "Виджет банковских операций"

## Цель проекта:

Цель проекта "Виджет банковских операций" - реализовать веб-приложение на Python для визуализации банковских операциях клиента, транзакциях.

## Установка и настройка в PyCharm:

1. Клонирование репозитория:
```
git clone https://github.com/username/project-x.git
```

2. Установка зависимостей:
```
pip install -r requirements.txt
```

3. Активация виртуального окружения:
```
python -m venv venv
source venv/bin/activate  # Для Linux/MacOS
venv\Scripts\activate     # Для Windows
```

4. Выполнение миграции посредством команды:
```
python manage.py migrate
```

5. Запуск локального сервера посредством команды:
```
python manage.py runserver
```

## Использование

### Модуль processing
Модуль processing содержит функции для обработки данных о транзакциях. 
Он находится в директории src/processing.py.

### Функции
#### Функция filter_by_state 
Функция принимает список словарей и опционально значение для ключа state (по умолчанию 'EXECUTED'). 
Функция возвращает новый список словарей, содержащий только те словари, у которых ключ state соответствует указанному значению.
#### Функция sort_by_date
Функция принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание). 
Функция должна возвращать новый список, отсортированный по дате (date).

1. Перейдите на страницу Вашего приложения,запустите приложение.

- функция принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание) и возвращает новый список, отсортированный по дате.
